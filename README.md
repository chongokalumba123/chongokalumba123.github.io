Resume placeholder

This folder contains a simple portfolio site (`index.html`). A placeholder `resume.pdf` was added so the "Download resume" button works.

To replace the placeholder with your real resume:

1. Create your resume as a PDF (A4 or US Letter, typically).
2. Name the file `resume.pdf`.
3. Place `resume.pdf` in the project root (next to `index.html`).
4. Commit and push to your repo.

Optional: If you want the download to open in a new tab instead of downloading, remove the `download` attribute from the link in `index.html`.

If you'd like, I can generate a nicer default resume PDF from your profile content and format it into a one-page PDF — say the sections you'd like to include (education, experience, skills)."
